Reconnaissance des PLL

Pour reconnaître rapidement quel PLL appliquer :

  1. Regardez les blocs de 1x2 sur les côtés (coins de même couleur adjacents)
  2. Comptez les "headlights" (deux stickers de même couleur côte à côte)
  3. Identifiez les coins résolus (même couleur sur 2 faces adjacentes)
  4. Pratiquez la reconnaissance en 2 looks : d'abord identifier la famille, puis le cas spécifique
