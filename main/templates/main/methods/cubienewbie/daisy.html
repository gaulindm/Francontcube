🎯 C'est quoi le but?

Faire une fleur avec 4 pétales blancs!

Les pétales blancs doivent entourer le centre jaune ☀️

🧩 Comment faire ?

  1. Trouve une arête blanche n'importe où sur le cube.
  2. Amène l'arête blanche sur la face jaune.
  3. Répète pour les 4 arêtes blanches.

Pour cette étape, aucun algorithme n'est requis. Utilise seulement tes intuitions et des rotations simples !

Avant

Arêtes blanches dispersées sur le cube

Après

Marguerite complète!

Clique sur les boutons pour voir comment la marguerite se construit étape par étape.

💡 Conseils utiles

  • Tourne le cube dans tes mains pour repérer les arêtes blanches.
  • Si une arête blanche est « coincée » en bas, utilise R U R' pour la libérer.
  • Ne t'en fais pas pour les couleurs autour – ce sera important à l'étape suivante !
  • Astuce : Commence par trouver UNE arête blanche, puis concentre-toi sur celle-là avant de chercher les autres.
